Display a different theme for each user level.

== Description ==

"Theme Tiers"¬ù is a new WordPress plugin created by Revive Media Services and available for free for download. It allows you to specify a different theme for each Wordpress user level.

== Installation ==

1. Upload `theme_tiers.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Theme Tiers' admin settings page and choose your theme settings

== Frequently Asked Questions ==

= What is an example usage of this plugin? =

A good example would be a blog or a news site that wanted to give different views or access permissions to different users. One could easily pair this theme with some sort of payment gateway, enabling 'upgraded viewing' of a site.
